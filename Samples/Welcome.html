Welcome to the Bing Maps V8 Code Sample project

This is a collection of code samples for the Bing Maps V8 web control which have been made open source on Github. These samples have been collected from a number of different sources. Some of these samples where created to assist developers on the Bing Maps forums many were created for the Bing Maps blog, MSDN Documentation and Interactive SDK while others were submitted by Bing Maps community developers. Be sure to also take a look at the list of external samples as well.
